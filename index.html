Hidden surface removal (HSR) algorithms

Z Buffering

zBuffering algorithm also called depth buffer algorithm is an algorithm designed to resolve the visibility problem, the principal idiea of this algorithm is while making rasteriation, to store the color and the depth in the pixel matrix, where the depth goes from 0(closest to the point of view) to 1(generally), this algorithm is usually implemented in hardware and OpenGL, the principal problem of this algorithm is that it requieres lots of memory, at least for a matrix WxL (number of pixel width and long).
In this algorithm to look for the depth of an image to be mapped to the projection plane, is used interpolation.

Advantages and Disadvantages


Advantages:

-Simple to use
-Easy to implement
-Fast at execution time
-Does not matter the order to rasterize the poligons, the result is always the same

Disadvantages:

-Uses a lot of memory
-Imposible to do transparent superfices with the original code.

Algorithm


Basic idea:


Rasterize every imput polygon
For every poligon find the deph (z value) by interpolation.
Save the smalles z or closest poligon in the pixels matrix.
After finish rasterizing every polygon, paint the color of every pixel, those should be the closest to the eye.

Pseudocode:


First initialice pixel Matrix(z_buffer) with 1 (maximum depth value)

For each polygon {
for each pixel (x,y) inside the polygon projection area {
if (z_polygon_pixel(x,y) < z_buffer(x,y) ) {
z_buffer(x,y) = z_polygon_pixel(x,y);
color_buffer(x,y) = polygon color at (x,y)
}
}
}
